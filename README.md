# Nytrw-Browser
![logo](Docs/logo.png)
A browser made with the Nytrw search engine

## Building
Run :
```bash
pip install PySide6 auto-py-to-exe
git clone https://github.com/nytrw/Nytrw-Browser.git
cd Nytrw-Browser
auto-py-to-exe
```
And select the **Nytrw-Browser.py** and use the options **onefile, noconsole** and optionaly choose the icon
